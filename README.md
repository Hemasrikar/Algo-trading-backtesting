# Algo-trading-backtesting

### Formulas 
 - Mid Price = $\dfrac{Bid + Ask}{2}$
 - Quoted Price = $Ask - Bid$
 - Relative Spread =  $\dfrac{Quoted Spread}{Mid Price}$
 - Effective Relative Spread = $\dfrac{2 |Price - Mid Price|}{Mid Price}$
 - Order  Imbalance = $\dfrac{Buy Volume - Sell Volume}{Total Volume}$
